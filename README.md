# locate

<!-- badges: start -->

<!-- badges: end -->

locate is a game that allows you to test your geography skills (in
France)

## Installation

The development version from [GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("VincentGuyader/locate")
```

## Example

``` r
library(locate)
locate()
```
